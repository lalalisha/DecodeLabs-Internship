## [1.0.0] — 2026-06-23

### Added
- Initial release of RuleBot
- 12 rule categories: greetings, identity, wellbeing, help, weather, jokes, thanks, motivation, fun facts, compliments, math, and emotional support
- Dynamic time/date responses via `datetime`
- Exit keyword detection with varied farewell messages
- Fallback response for unmatched input
- Unit test suite (`tests/test_rulebot.py`)
- GitHub Actions CI workflow (Python 3.8 / 3.10 / 3.12)
